## 🎯 算法 4 维加权 λ (v2.1)

- 位置分桶：FW=3 / MID=3 / DEF=4 / GK=1
- 球员权重：0.7 / 教练：0.3
- `λ_home = 1.3 + (attack_home - defense_away) × 1.5`
- 平局：15% / 加时：强队 λ 优势 / 点球：按胜率比
